# Demo Seed Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Seed two identical demo buildings with 15 tenants, Jan–Mar historical transactions, and generate April + May Excel files for live upload during customer demo.

**Architecture:** Single Python script (`backend/seed_demo.py`) connects via DATABASE_URL, deletes junk test buildings, inserts 2 identical buildings (one for practice, one for the real demo), seeds all data, then generates 2 Excel files into `demo_files/`.

**Tech Stack:** Python 3.11, SQLAlchemy (raw `text()` inserts), python-dotenv, openpyxl, pandas
